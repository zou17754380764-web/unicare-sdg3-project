# UniCare – Campus Healthcare Appointment System

UniCare is a software-based prototype designed to improve access to campus healthcare by simplifying how students book and manage clinic appointments.

SDG Goal:
SDG 3 – Good Health and Well-being

Development Methodology:
Agile Scrum

Tools Used:
- Trello (task management)
- GitHub (version control)

Project Structure:
planning – project planning documents
design – system diagrams
prototype – UI prototype
testing – testing documentation
report – final report
